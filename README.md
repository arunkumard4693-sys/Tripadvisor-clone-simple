# TripAdvisor Clone

Welcome!
This is a simple TripAdvisor homepage clone built using only HTML and CSS.

## About the Project

The project is a front-end clone of Tripadvisor created for learning and practicing basic web development skills. It focuses on layout design and styling without using JavaScript or any frameworks.

## Built With

- HTML
- CSS

## Features

- Navigation bar
- Destination/recommendation cards
- Footer section
- Clean and simple UI
